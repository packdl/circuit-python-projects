# Circuit Playground  Ornament
This is a simple Circuit Playground ornament inspired by this [Adafruit Learn Guide](https://learn.adafruit.com/circuit-playground-tft-gizmo-snow-globe). It's a fairly simple design.

## Usage
The A button turns on the display.   
The B button turns off the display.   
Turn the ornament upside down for 3 seconds, and the display turns off.
